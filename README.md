# BMI Calculator

A simple and responsive BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript. This application allows users to calculate their BMI based on height and weight and displays the corresponding weight category.

## 📖 How BMI is Calculated

BMI is calculated using the formula:

```text
BMI = Weight (kg) / Height² (m²)
```

### BMI Categories

| BMI Range | Category |
|------------|------------|
| Below 18.5 | Underweight |
| 18.5 - 24.9 | Normal Weight |
| 25 - 29.9 | Overweight |
| 30 and Above | Obese |
